# Initialise project

Run the following command:

```
  npx create-react-app boilerplate-0
```

Open folder boilerplate-0 with VS Code.

Initialise git and publish master.

```
  git branch dev
  git checkout dev
```

Publish dev branch.

Test the react app:

```
  npm start
```

Install React router

```
  npm install --save react-router-dom
```
